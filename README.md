# Lambda API template

A Lambda API based on [Serverless](https://serverless.com/).

## How to deploy

Install the dependencies:
```
npm install
```

Deploy the dev environment:
```
npm run deploy
```

Check the logs:

```
npm run logs <NAME_OF_FUNCTION>
```
For production:
```
npm run logs-prod
```

## Create an IoT role

```
npm run create-iot-role
```
